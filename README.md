# KOReader Komga Client Plugin

A KOReader plugin for Komga servers. It provides a native browser interface and synchronizes reading progress.

## Working Features
- Native Komga browser UI
- Browse libraries, series, recently added, and on-deck books
- Filter series by read status (Unread, In Progress, Completed)
- Direct book downloads with metadata extraction
- Progress sync (forward/backward) on book open/close

## Installation
Download and copy `kokomga.koplugin` folder to KOReader `plugins` folder

## Configuration
1. Open KOReader's top menu
2. Go to the Search tab (magnifying glass icon) -> kokomga
3. Enter your Server URL (e.g., `http://192.168.1.100:8080`)
4. Enter your API Key
5. Open "Komga Browser (New)" to view your library
